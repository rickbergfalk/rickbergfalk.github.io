Interactive SQL Lessons (2011 - 2014)

Ready Set SQL is a series of interactive SQL lessons geared towards learning SQL useful for reporting purposes. The target audience is semi-tech-savvy business people who may have access to some sort of database (either directly or via some tool such as Microsoft Access).

It's mostly finished. Just needs more content and some final tweaking.
