Ready Set SQL

Interactive SQL Lessons (2011 - 2013)

Ready Set SQL is a series of interactive SQL lessons geared towards learning SQL useful for reporting purposes. The target audience is semi-tech-savvy business people who may have access to some sort of database (either directly or via some tool such as Microsoft Access).

It's almost finished. Just needs more content and some final tweaking.

Chart Builder

UI Prototype (2010)

An experimental UI for creating charts and graphs. The primary focus was to create a tablet friendly UI and a simple approach to selecting a chart and data.

The main interaction was driven via the isotope plugin for jQuery. I didn't get very far with it, partially because coding this kind of UI got confusing, and the interaction/usability wasn't very fitting for its use case. Sure it looked cool, but how practical would it have been?

Ugly Dashboard Builder

UI Prototype (2009)

A drag and drop interface experiement to create ugly dashboards using the Google Vis API. A self-challenge utilizing jQuery, object-oriented JavaScript, Google Visualization API, and fancy CSS.
